# AKOS Core Roadmap

## 0.1 — Portable career profile

- JSON career source data
- starter portfolio
- configurable Buy Me a Coffee support link
- self-hosting instructions

## 0.2 — Resume builder

- role and vacancy input
- reusable resume templates
- Markdown and PDF output
- selection of relevant projects and capabilities

## 0.3 — Evidence engine

- connect every generated claim to source evidence
- confidence scoring
- unsupported-claim warnings
- claim exclusions and boundaries

## 0.4 — Import and onboarding

- import an existing CV
- guided career interview
- project and achievement editor
- multilingual profiles

## 1.0 — Open career operating system

- portfolio, CV, cover letter and interview outputs
- optional managed AKOS Cloud services
- template marketplace
- consultant and team workflows
