# Deploy your AKOS site

## Fastest path: GitHub + Vercel

1. Open the AKOS Core repository.
2. Click **Use this template** and create a new repository in your GitHub account.
3. Edit `career-data/profile.json` with your own career data.
4. Open Vercel and choose **Add New → Project**.
5. Import your new GitHub repository.
6. Keep the default Next.js settings and click **Deploy**.
7. Vercel will give you a public URL. Every future GitHub commit will trigger a new deployment.
